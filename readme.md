## Katomia

A real-time coordination engine for multi-party value exchange — enabling atomic, multi-asset swaps on Hedera.

👉 Try the demo: https://katomia.com

In the beginning, exchange was human — based on trust, relationships, and context.

Modern systems collapsed this into a single dimension: money.

Katomia explores what happens when coordination and trust return — powered by real-time systems and distributed consensus.

### The Problem

Money solved coordination — specifically the *coincidence of wants*.

But it reduced value to a single dimension.

Modern systems (including DeFi) largely continue this model, relying on price, pairs, and liquidity pools — limiting more complex forms of exchange.

### The Solution

> From a tiny crack in the rock new growth forms, initially un-noticed, but then as roots set the rock is broken apart from the inside.

Katomia enables direct coordination instead of price-based exchange.

Participants propose direct asset transfers between each other, forming dynamic trade graphs in real time. Once agreement is reached, Katomia composes and executes a single atomic transaction on Hedera.

No intermediaries. No order books. No pair constraints.

### How it works

1. Create a coordination space (“game”)
2. Participants join with accounts
3. Propose transfers by placing assets into others’ baskets
4. Accept, reject, or modify proposals in real time
5. Lock when agreement is reached
6. Sign & execute a single atomic transaction on Hedera

Katomia supports flexible agreement models:

- **All**: everyone must agree  
- **Active**: only participants in the trade  
- **Cluster**: independent groups can settle separately  

### Why Katomia is different

Katomia replaces price-based exchange (order books and AMMs) with direct coordination.

Instead of trading via pairs or liquidity pools, participants construct multi-party exchange graphs that are executed atomically.

This enables coordination that traditional DeFi cannot express.

> Katomia is not a marketplace — it's infrastructure for new kinds of markets
